Raspberry Pi Imagerを使い書き込んだSDカード上のRaspberry Pi OSで、このスクリプトを実行するとABSのPBXとなります。QPMも含む全機能(addon機能も)が使用可能となります。

なお、インストールを開始する前にロケール、タイムゾーン等の設定を行っておいてください。GUI(Desktop)は必要ありませんのでCLIログインにしておいてかまいません。

ABSのWebGUIには http://Raspberry Pi のIPアドレス でアクセスできます。qpmには http://Raspberry PiのIPアドレス/qpm でアクセスできます。なお、QPMを使用するにはQPMのユーザ登録が必要となりますので、ABS WebGUIから登録を行っておいてください。
